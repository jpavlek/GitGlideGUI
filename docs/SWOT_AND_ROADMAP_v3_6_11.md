# SWOT and Roadmap v3.6.11

## Strengths
- Better branch-context visibility.
- Stronger protected-branch guardrails.
- Clearer local-first Git workflow identity.

## Weaknesses
- Main WinForms script remains large.
- PSScriptAnalyzer warnings are still curated rather than fully fixed.

## Opportunities
- Guided Merge & Publish checklist.
- Quality-check evidence before main promotion.
- Optional GitHub CLI / PR support.

## Threats
- Mature Git clients remain more polished.
- Trust can be lost quickly if package/version metadata drifts again.
