# SWOT and Roadmap v3.8.0

## Strengths

- Stronger branch decision support before risky operations.
- Read-only relationship checks reduce accidental side effects.
- Builds directly on the v3.7 split-script foundation.

## Weaknesses

- The graph is still not a fully graphical commit DAG.
- Branch relationship UI is still script-based rather than extracted into a dedicated panel module.

## Opportunities

- File history and commit detail workflows.
- Visual merge conflict assistant.
- Interactive rebase and commit cleanup.

## Next roadmap

1. v3.8.1: file history and selected commit detail polish.
2. v3.9.0: visual merge conflict assistant.
3. v3.10.0: interactive rebase and commit cleanup.
