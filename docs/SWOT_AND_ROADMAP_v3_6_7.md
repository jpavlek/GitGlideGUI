# SWOT and Roadmap - v3.6.7

## Strengths

- More transparent GitHub remote setup.
- Safer distinction between deleting a file and stopping Git tracking.
- Better local-first workflow for human and AI-assisted iterations.

## Weaknesses

- GitHub repository creation is still manual unless the user uses the browser.
- Remote diagnostics depend on local Git authentication.

## Opportunities

- Add optional GitHub CLI integration.
- Add PR/compare helpers.
- Add `.gitignore` recommendations after stop-tracking.

## Threats

- Mature Git clients already include broad remote-host integration. Git Glide should stay focused on transparency, judgment, and safer local workflows.
