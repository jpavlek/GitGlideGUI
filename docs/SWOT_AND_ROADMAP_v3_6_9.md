# Git Glide GUI v3.6.9 SWOT and Roadmap

## Strengths

- Local-first, previewable Git actions.
- Restored explicit Git Flow merge path.
- Safer developer control for human and AI-assisted iterations.

## Weaknesses

- Merge workflow is still command-based, not a full visual branch graph assistant.
- GitHub repository creation still requires browser/manual setup.

## Opportunities

- Add GitHub CLI support for private repo creation.
- Add release tagging and branch cleanup after promotion to main.
- Add a clearer visual timeline for feature/develop/main promotion.
